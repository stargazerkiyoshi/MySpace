## 目的
定义 Space 内节点关系的持久化约束，为 Graph View 的显式连线编辑提供主规格。

## 需求
### 需求: 系统必须持久化 Space 内节点关系
系统必须为每个 Space 持久化节点之间的显式关系，以便 Graph View 可以在刷新后保持用户编辑过的连线结构。

#### 场景:创建节点关系
- **当** 用户在某个 Space 内为两个不同 Node 创建连线
- **那么** 系统必须持久化一条关联这两个 Node 的关系记录
- **并且** 同一对方向相同的节点关系不得重复创建

#### 场景:删除节点关系
- **当** 用户删除某个 Space 内已有的节点关系
- **那么** 系统必须删除对应关系记录
- **并且** 后续读取图数据时不得再返回这条关系

#### 场景:禁止无效关系
- **当** 用户尝试把一个 Node 连到自己，或把不属于当前 Space 的节点建立关系
- **那么** 系统必须拒绝该请求
- **并且** 不得写入无效关系数据
