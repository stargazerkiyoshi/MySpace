## ADDED Requirements

### 需求:本地依赖服务编排
系统必须提供 `docker-compose.yml` 或等效本地编排配置，以启动 PostgreSQL 和 Redis 等本地开发依赖。

#### 场景:准备本地开发依赖
- **当** 开发者按照文档启动本地基础设施
- **那么** 系统必须能够启动前端与服务所依赖的数据库和缓存服务

### 需求:环境变量模板
系统必须提供 `.env.example` 或等效模板，以声明 `web`、`core-api`、`ai-service` 与本地基础设施所需的关键环境变量。

#### 场景:初始化本地环境变量
- **当** 开发者参考环境变量模板配置本地环境
- **那么** 开发者必须能够识别每个应用的核心配置项和默认本地依赖地址

### 需求:统一启动说明
系统必须提供 README 启动说明，明确 monorepo 安装步骤、依赖服务启动方式和各应用的运行命令。

#### 场景:新开发者接入项目
- **当** 新开发者首次阅读项目 README
- **那么** 文档必须能够指导其完成依赖安装、本地基础设施启动和三个应用的开发启动
